Flow Types

Creativity

Spark

Business Process Flows

Step-by-step guide for your users to get the work done in a consistent way. Through its stages, guides the users through the required steps to complete a task.

Desktop Flows

Designed for existing web or desktop applications. You can record interactions, and through the user interface, interact with any application. Specially useful for legacy programs that can not be automated in any other way.

Cloud Flows

You want to use it when you have exposed services or APIs and want to interact with them.
