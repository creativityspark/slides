What is Microsoft Dataverse?

Dataverse can be summarized as a Database in the cloud.

But is much more ...

It lets you securely store and manage data used by Business Applications.

Data within Dataverse is stored within a set of Tables.

Creativity

Spark

Dataverse includes a base set of standard tables that cover typical scenarios, but you can also create custom tables specific to your organization.

App makers can then use Power Apps to build rich applications using this data.

Or you can create your own front-end to manipulate the data.

A Table is a set of Rows and Columns.

Common Data Service and Microsoft Dataverse are the same thing.

Microsoft recently renamed the Common Data Service to Microsoft Dataverse. Also renamed other parts of the application.

For example, Entity is now Table and Attribute is now Column.

Power Platform is a family of products based on the low-code philosophy.

Creativity

Spark

Power Apps is one of the products in the family specifically designed for building User Interfaces.

Microsoft Dynamics 365 is a family of Business Applications built on top of Power Platform and Dataverse.

Creativity

Spark

Low-code means that little to no code is required to build an application.

Creativity

Spark

A Citizen Developer is a person with no coding skills that is able to use the functionality provided by the Power Platform to build secure and reliable enterprise applications.

Think for example of those people in your organization that use their Excel skills to build all sort of spreadsheet trackers, or workbooks full of formulas to calculate budgets.

Now these people are Citizen Developers in the Power Platform.

How can Power Platform help my organization?

Creativity

Spark

Power Platform makes it easier to quickly build Business Applications. And quicker means less time, less resources and cheaper.

This approach where anyone within the company with Microsoft Office skills could model an application that automates parts of your business processes in a managed, secure and reliable way is a game-changer.

Now the old budget spreadsheets can be turned into great apps in a controlled and safe way.
